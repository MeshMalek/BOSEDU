# Bosedu App

## Overview
Bosedu is a Flutter application designed for business education and networking. The app includes onboarding screens that introduce users to the platform and its features.

## Screens
The app currently consists of the following five screens:

1. **Splash Screen**: A visually appealing splash screen displaying the app logo.
2. **Onboarding Screen 1**: Introduction to the product and its benefits.
3. **Onboarding Screen 2**: Information on events and how to join them.
4. **Onboarding Screen 3**: Details about educational resources available.
5. **Onboarding Screen 4**: Descriptions of private discussion features.
